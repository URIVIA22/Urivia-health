App icon reference sheet — all required iOS and Android sizes
How to export:

Screenshot each icon at the exact pixel size shown, or use this page as a reference for your designer. The SVG mark below scales perfectly to any size. For production, export the 1024×1024 version and let Xcode / App Store Connect auto-generate smaller sizes using the AppIcon.appiconset Contents.json.

Production export: Save as AppIcon.appiconset in Xcode. Upload only the 1024×1024 to App Store Connect — it generates all other sizes automatically. For Android, save as ic_launcher.png in the appropriate mipmap-* directories. All icons must have no transparency on iOS.
